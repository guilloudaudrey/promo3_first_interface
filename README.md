# Math API

A simple API to do math calculation.

## Addition

Realize an addition.

* __URL__: 

    /add

* __Method:__

    GET

* __URL Param:__

    __require:__
    
    `a=[number]`

    `b=[number]`

* __Success Response:__

    __Code:__ `200 OK`
    
    __Content:__ `[number]`

* __Error Response:__

    __Code:__ `400 Bad Request`

    __Content:__ None
